# Kymarysse Nail Salon Website

## Description
This project is a responsive front-end website for Kymarysse Nail Salon. It showcases the salon’s services and includes an interactive feature that allows users to update the call-to-action (CTA) headline dynamically.

The goal of this project is to demonstrate fundamental web development concepts using HTML, CSS, and JavaScript.

## Features
- Clean and modern landing page design
- Navigation bar with section links
- Hero section with a call-to-action
- Services section highlighting offerings
- Interactive form to update the CTA headline
- Contact information in the footer
- Responsive design for different screen sizes
- Styled UI with gradients, hover effects, and modern layout elements

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM manipulation)

## File Structure
- `index.html` – Main HTML structure  
- `style.css` – Styling for layout, colors, responsiveness, and UI effects :contentReference[oaicite:0]{index=0}  
- `cc_7.js` – JavaScript for interactivity (CTA update)

## How It Works
1. The user enters a new headline in the input field.
2. When the button is clicked, JavaScript updates the CTA text in the hero section.
3. The page reflects the change instantly without reloading.

## How to Run
1. Download or clone the repository  
2. Open `index.html` in a web browser  
3. Interact with the CTA update feature to see dynamic changes  

## Concepts Demonstrated
- Semantic HTML structure  
- Responsive layout design  
- CSS styling (including gradients, flexbox, grid, and media queries)  
- DOM selection and manipulation  
- Event handling in JavaScript  
- User input handling  

## Notes
- This is a front-end only project (no backend or database)  
- The CTA update is handled entirely on the client side  
- The styling includes responsive breakpoints for mobile and tablet views  
- Designed as a functional UI demonstration with modern design practices
